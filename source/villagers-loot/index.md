# 村民与战利品改动

> [绿宝石] RSMP S2 调整了村民交易和装备获取路线。装备不再能轻松从普通村民处获得。

## 普通村民限制

普通村民禁售：

- 附魔书
- 钻石装备

## 傻子村民

傻子村民会变成：

> 建材商店

出售大量建筑材料。

具体商品列表：待补充。

## 战利品改动

| 内容 | 说明 |
|---|---|
| 陶片 | 可从钓鱼宝箱等来源获得 |
| 附魔书 | 可从钓鱼宝箱等来源获得 |
| Boss 奖励 | 包含回响碎片、下界合金材料、技能信息残页、祭坛残页 |
| 其它战利品 | 待补充 |
