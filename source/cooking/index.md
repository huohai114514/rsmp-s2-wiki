# 炒菜系统

> [熔炉] RSMP S2 的炒菜不是工作台合成。你需要在熔炉或烟熏炉上真正“翻锅”。

## 玩家怎么操作

| 动作 | 操作 |
|---|---|
| 投入食材 | Shift + 右键熔炉或烟熏炉 |
| 最多食材 | 5 份 |
| 开始翻炒 | 持剑右键熔炉 |
| 继续翻锅 | 持剑右键熔炉 |
| 烹饪时长 | 约 9 秒 |
| 出锅拿取 | Shift + 右键熔炉 |

烹饪时，ActionBar 会显示火候条。

<div class="actionbar">火候 ▓▓▓▓▓░░░░░ 约 48% - 70% 为理想区间</div>

翻锅次数接近 **7 次** 时评分较好。

## 品质等级

| 品质 | 说明 |
|---|---|
| 普通 | 基础菜品 |
| 稀有 | 更好的补给 |
| 史诗 | 高品质菜品 |
| 传说 | 顶级菜品 |

品质会影响：饱食度、饱和度和份数。

拥有料理相关能力的玩家会做出 **珍馐**。珍馐最低为稀有，评分额外提高，食物与饱和倍率更高。

## 奖励 / 风险

| 类型 | 内容 |
|---|---|
| 奖励 | 高品质菜品、Buff、更多份数 |
| 风险 | 过冷、过热、翻锅节奏错误会降低品质 |
| 高风险食材 | 异味食材可能带来负面效果 |
