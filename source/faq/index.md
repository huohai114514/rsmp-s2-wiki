# FAQ 常见问题

## 这是官网吗？

这是 RSMP S2 玩家资料站，用来介绍服务器玩法、插件机制、入服教程和更新内容。

## 这个网站需要服务器后端吗？

不需要。这个 Wiki 是 Hexo 生成的静态网站，部署到 GitHub Pages 后不需要数据库和后端。

## 为什么我看不到 TAB 列表？

RSMP S2 为了隐私和剧情体验，普通玩家可能无法查看 TAB 列表和玩家名补全。

## 定位条会被隐藏吗？

定位条应该保留可见。如果你发现定位条不可见，请反馈给管理员。

## 技能数值一定准确吗？

以服务器当前插件版本为准。Wiki 内容会随着 SoulHeadsFolia 热修版本更新。
