# SoulHeadsFolia 总览

SoulHeadsFolia 是 RSMP S2 的核心玩法插件之一，围绕灵魂、头颅、技能、复活和玩家互动展开。

## 核心玩法

| 模块 | 说明 |
|---|---|
| 灵魂头颅 | 玩家死亡、绑定、复活等玩法核心 |
| 技能系统 | 不同技能拥有不同触发方式和冷却 |
| 技能抽取书 | 用于抽取或查看技能相关内容 |
| 技能图鉴 | 给玩家查看所有技能说明 |
| 复活祭坛 | 与复活、献祭、头颅相关的特殊结构 |
| 排行榜 | 记录击杀、死亡、钓鱼、钻石块、种地等数据 |
