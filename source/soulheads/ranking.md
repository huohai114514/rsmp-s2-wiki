# 排行榜

RSMP S2 的排行榜用于记录玩家生涯累计数据和赛季荣誉。

## 统计项目

| 排行榜 | 说明 | 头衔 |
|---|---|---|
| 击杀 | 玩家击杀累计 | JohnWick |
| 钻石块 | 背包 + 末影箱统计 | Mr.Diamond |
| 钓鱼 | 钓鱼累计 | Santiago |
| 死亡 | 死亡次数累计 | UGottaMove |
| 种地 | 农业相关累计 | Mr.Bean |

## 显示规则

- 排行榜为生涯累计数据。
- 每周公布榜一并发放对应头衔。
- 公布前仅显示排名，避免提前暴露完整数据。
- UI 建议纵向排列，方便玩家快速查看。
