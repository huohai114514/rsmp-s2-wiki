# 命令说明

这里用于记录玩家可用命令和管理员命令。正式上线前请按插件实际命令修改。

## 玩家命令

| 命令 | 说明 | 权限 |
|---|---|---|
| `/skills` | 打开技能相关界面 | 玩家 |
| `/ranking` | 查看排行榜 | 玩家 |
| `/wiki` | 查看 Wiki 链接 | 玩家 |

## 管理员命令

| 命令 | 说明 | 权限 |
|---|---|---|
| `/soulheads reload` | 重载插件配置 | OP |
| `/soulheads debug` | 调试插件状态 | OP |

## 隐私与显示规则

| 项目 | 说明 |
|---|---|
| TAB 补全 | 普通玩家不应看到玩家名补全 |
| TAB 列表 | 普通玩家隐藏 |
| 定位条 | 普通玩家应保留可见 |
| 主菜单在线信息 | 对普通玩家隐藏 |
