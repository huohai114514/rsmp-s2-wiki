# 精英怪与 Boss

> [BossBar] 荒野里，有些怪物不是普通怪。高价值生物可能突变为精英怪，精英怪还可能进一步成为 Boss。

## 精英怪

精英怪前缀：

> [精英]

精英怪强化：

| 属性 | 倍率 |
|---|---:|
| 生命 | 1.5x |
| 伤害 | 1.2x |

可能突变的生物：守卫者、远古守卫者、监守者、劫掠兽、猪灵蛮兵、末影龙、凋灵。

## Boss 突变

精英怪有 10% 概率进一步突变为 Boss。

| 项目 | 数值 |
|---|---:|
| 战斗场地半径 | 35 格 |
| 单世界最多 Boss | 3 个 |
| 生命 | 精英基础上 8x |
| 伤害 | 2x |
| 速度 | 1.12x |
| 击退抗性 | 0.75 |

Boss 会拥有 BossBar、专属竞技场、技能冷却和奖励掉落。

## Boss 奖励

可能包括：附魔之瓶、回响碎片、下界合金碎片、下界合金锭、技能信息残页、祭坛残页。
