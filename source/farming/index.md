# 农务与果树系统

> [堆肥桶] RSMP S2 的农务不是简单骨粉催熟。肥料类型、作物偏好和品质评分都会影响收成。

## 玩家怎么操作

### 制作肥料

玩家可以通过堆肥桶制作三种肥料。

| 肥料 | 外观 | 来源 |
|---|---|---|
| 氮肥 | 绿色染料 | 动物肉类堆肥 |
| 磷肥 | 白色染料 | 骨头或骨粉堆肥 |
| 钾肥 | 橙色染料 | 植物、果实类堆肥 |

### 给作物施肥

手持肥料右键作物或树苗即可施肥。

正确施肥会加速生长并提高品质分。错误施肥会降低品质分。连续错肥或评分过低会导致作物枯死。

## 作物品质

成熟作物被破坏时，不掉原版普通作物，而掉落 RSMP S2 农务产物。

| 品质 | 分数要求 |
|---|---:|
| 普通 | 低于 28 |
| 稀有 | 28 分 |
| 史诗 | 58 分 |
| 传说 | 82 分 |

## 奖励 / 风险

| 类型 | 内容 |
|---|---|
| 奖励 | 高品质农务产物、出售绿宝石、用于炒菜 |
| 风险 | 错肥会降分，连续错误可能枯死 |
| 经济价值 | 品质越高，出售越值钱 |

## 农务产物出售

手持农务产物右键堆肥桶可以出售换绿宝石。

| 品质 | 绿宝石 |
|---|---:|
| 普通 | 1 |
| 稀有 | 3 |
| 史诗 | 5 |
| 传说 | 8 |

# 果树系统

> [树苗] 给树苗施肥后，它会被登记为果树。

## 果树怎么工作

- 对树苗施肥会登记为果树。
- 果树长成后，会寻找树叶下方的空位生成悬挂果实展示物。
- 右键果实展示物即可采摘。
- 果实最多保留一段时间，长时间未摘会掉落并降低品质。
- 最多同时挂 4 个果实。
- 正确施肥会提前结果。
- 生物群系契合会提高果实品质，错误群系会降低品质。
