# 加入 RSMP S2

本页用于给新玩家说明如何加入服务器。正式上线前，请把这里的 IP、端口、版本要求改成你的实际配置。

## Java 版

| 项目 | 内容 |
|---|---|
| 推荐版本 | Minecraft Java 1.21.11 |
| 服务端 | Folia |
| 地址 | `请填写你的服务器 IP` |
| 端口 | `25565` 或你的实际端口 |

## 基岩版

RSMP S2 支持 Java + 基岩版互通时，可以在这里写 Geyser 入服信息。

| 项目 | 内容 |
|---|---|
| 地址 | `请填写你的服务器 IP` |
| 端口 | `请填写 Geyser 端口` |
| 登录方式 | Floodgate / 正版验证，按服务器实际配置填写 |

## 新玩家建议

1. 先阅读 [FAQ](/rsmp-s2-wiki/faq/)。
2. 进入服务器后确认规则和玩法限制。
3. 查看 [技能图鉴](/rsmp-s2-wiki/soulheads/skills/) 了解核心玩法。
4. 不要随意破坏祭坛、复活结构和公共建筑。
