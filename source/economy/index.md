# 经济与附魔

本页用于整理 RSMP S2 的经济、附魔、铁砧、村民和装备规则。

## 经验与铁砧

| 项目 | 规则 |
|---|---|
| 等级上限 | 39 级 |
| 每级经验 | 120 |
| 铁砧费用 | 线性计算，最多 50 经验 |

## 装备与附魔

| 项目 | 规则 |
|---|---|
| 下界合金甲 | 禁止修补 |
| 村民交易 | 禁售附魔书与钻石装备 |
| 附魔台 | 装备 + 青金石 + 回响碎片 + 陶片 + 矿物块 + 经验 |
| 陶片 | 决定附魔种类 |
| 矿物块 | 决定附魔等级 |
