# RSMP S2 简介

RSMP S2 是一个以生存、灵魂、头颅、技能、祭坛和玩家故事为核心的 Minecraft 生存服赛季。

## 赛季关键词

- 生存探索
- 玩家互动
- 灵魂头颅
- 技能系统
- 复活祭坛
- 排行榜
- 料理、农业、钓鱼
- Java 与基岩版互通

## Wiki 的作用

这个 Wiki 用来给玩家快速查询玩法规则，不需要玩家在群里反复问：

- 某个技能怎么触发？
- 技能冷却多久？
- 技能抽取书怎么合成？
- 复活祭坛怎么用？
- 排行榜统计什么？
- 基岩版怎么进服？

## 内容状态

本 Wiki 是静态站点，内容以服务器实际插件版本为准。管理员更新插件后，应同步更新这里的说明。
